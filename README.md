# Student Performance Prediction Machine Learning Project

## Overview

This project uses Machine Learning to predict student performance based on various academic features. The goal is to create a predictive model that can help educators and institutions identify students at risk of underperforming and take proactive measures to improve their outcomes.

The model is deployed using **Flask** to create an interactive web application where users can input data and get predictions about student performance.

## Features

- **Predictive Model**: A machine learning model that predicts student performance based on input features such as study time, attendance, and past grades.
- **Interactive Web Interface**: Built using **Flask**, this web app allows users to enter student data and see real-time predictions.
- **Data Visualization**: Displays important insights about the model's predictions and helps analyze student performance.
- **Model Deployment**: Deployed locally or on cloud platforms for easy access.


## Technologies Used

- **Python**
- **Flask** (for web app deployment)
- **scikit-learn** (for machine learning model)
- **catboost** (for advanced gradient boosting)
- **xgboost** (for optimized gradient boosting)
- **numpy** (for numerical calculations)
- **pandas** (for data manipulation)
- **seaborn** (for data visualization)
- **dill** (for serializing the model and saving/loading it efficiently)
